README

Manager.h
    Imports all manager interfaces in this sub folder.
    
KNCommunicationManager
    Manager of communication between local device and remote server
    
UserManager
    Manager for user information
    
CoreDataManager
    Manager for core data.

KNAppControlManager
    Manager for application control which controls various switches and so on.
    
KNStoryboardManager
    Manager navigations in storyboard based on switches and status.
