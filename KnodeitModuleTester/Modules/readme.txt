OUR COMPLETE APPLICATION IS DIVIDED INTO MODULES


The following modules are available:

BaseModule
    contains generic methods that can be subclassed

Contacts
    contains methods for addressbook
    
AuthenticationModule
        MobileRegisteration : register phone number template
        Login View          : login template
        Registration        : registeration template
        
CameraPickerModule
    view template which includes camera picker
    
FriendPickerModule
    view template which includes table view for friend list.
    
InboxModule
    view template for inbox

SplashModule
    view template for splash
